Website, Web browser, Webpage, Web Server, URL

Website: A website is a collection of web pages that are related to each other and can be accessed over the internet.

Web browser: A web browser is a software application used to access and display websites on the internet. Examples include Chrome, Firefox, and Safari.

Webpage: A webpage is a single document or page within a website. It contains information such as text, images, and multimedia elements.

Web Server: A web server is a software or hardware that stores, processes, and serves website content to users over the internet.

URL: A Uniform Resource Locator (URL) is a web address that specifies the location of a resource on the internet. It consists of a protocol, domain name, and path.
